|---|---|---|

## Production notes

- Publish day: **Tuesday at 10 AM ET** (matches the CRK side and aligns with MTGGoldfish + Star City weekly cadence).
- Set tier lists: ship within **7 days of set release**. Search demand is concentrated in week 1.
- Standard meta refreshes: ship within **48 hours of B&R announcement**. The next scheduled B&R is June 30, 2026.
- Reuse the `crk-blog-writer` agent prompt scaffold (saved during launch) adapted for MTG — same voice spec, same TOC verifier, same accuracy guardrails.
- Always run `node scripts/verify-blog-toc.js` before commit — it now scans both `cookie-run-kingdom/` and `magic-the-gathering/` post folders.
- For any post that touches current prices, ban legality, tournament results, or rules text: mark `[Verify current price]` / `[Verify current legality]` / `[Verify rules text]`. Better to under-claim than invent.
