## Production notes

- Default to publishing on **Tuesdays at 10 AM ET**. This is when search demand for new-week CRK content peaks.
- Time-sensitive posts (patch overviews, event guides) should publish within 48 hours of Devsisters' announcement — race condition with PocketGamer / Game8.
- Reuse the writer-agent prompt scaffold (saved during launch) for future posts; it reliably produces voice-consistent drafts.
- Run the `Optimize` agent (or equivalent editor pass) before publishing any time-sensitive post — too easy to ship a fact-flagged claim.
