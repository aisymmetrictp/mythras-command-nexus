## Cadence rules (apply to every entry)

- **Monthly tier lists:** prefer ONE canonical evergreen slug (`cookie-run-braverse-tier-list`) that you re-date in the H1 and bump via `lastUpdated`, OR dated slugs as above if you want each month archived. Pick one approach and stay consistent — don't fragment ranking authority across many thin URLs.
- **Set reviews:** ship in release week; do a 30-day "now that the meta's settled" refresh via `lastUpdated`.
- **Per-color card guides + per-archetype deck guides:** refresh each time a new set meaningfully changes the color/archetype; bump `lastUpdated` only.
- **Event coverage:** only publish confirmed, verified results. Never pre-write a winner.
- **Every entry inherits the strategy doc's `[Verify]` discipline.** When in doubt, flag it.
